# Excel Library Evaluation

- مكتبة xlsx تدعم القراءة والكتابة والصيغ الأساسية، لكنها لا تدعم كتابة صيغ XLOOKUP/VLOOKUP/INDEX/MATCH بشكل كامل في بعض الإصدارات القديمة.
- مكتبة exceljs تدعم إنشاء ملفات متعددة الأوراق، وتنسيق الخلايا، وكتابة صيغ Excel، لكنها لا تنفذ الصيغ (تُترك للتطبيق).
- exceljs أفضل للتنسيق المتقدم، لكنها أبطأ في بعض العمليات.
- حالياً xlsx كافية للوظائف المطلوبة (كتابة صيغ، أوراق متعددة، دعم SUM/IF/VLOOKUP/INDEX/MATCH/COUNTIF/SUMIF كصيغ نصية). إذا احتجنا تنسيق متقدم جداً أو حماية أوراق، يمكن إضافة exceljs لاحقاً.

## التوصية
- الاستمرار مع xlsx حالياً.
- إذا ظهرت متطلبات تنسيق متقدمة جداً أو حماية أوراق، أضف exceljs بجانب xlsx واستخدمها فقط عند الحاجة.
